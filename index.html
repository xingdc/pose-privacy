一、隐私政策声明（中文）

Pose摄影助手（以下简称「本应用」）是一款纯离线拍照姿势参考工具，本隐私政策用于向用户清晰说明数据使用、隐私保护及版权相关规则。

1. 信息收集

本应用不会收集、上传、共享任何用户个人信息。应用无相机、相册、定位、麦克风、通讯录、网络等敏感权限，全程不会获取用户任何私人数据。

2. 本地数据存储

用户在应用内收藏的姿势数据，仅保存在用户手机本地设备中，不会上传至任何云端服务器。用户卸载本应用后，所有本地收藏数据将被永久清除，且无法恢复。

3. 网络使用说明

本应用无需联网、无任何网络请求，全部姿势资源内置在安装包内，支持全程离线使用，不会产生任何网络流量消耗。

4. 第三方服务

本应用未接入任何第三方SDK、广告插件、数据统计工具，不存在任何第三方数据收集、推送、广告展示行为。

5. 素材版权声明

本应用内所有姿势插画均为开发者原创AI绘制素材，无真人出镜、无网络盗用图片、无第三方版权内容，完全规避肖像权与版权侵权风险，仅作个人日常参考使用。

6. 用户权利

用户可随时自由删除、卸载本应用，设备内不会留存任何个人使用痕迹。若对产品有疑问、反馈或功能建议，可通过开发者邮箱进行沟通。

二、Privacy Policy (English)

Pose Photography Assistant (hereinafter referred to as "the App") is a pure offline photography pose reference tool. This privacy policy clearly explains data usage, privacy protection and copyright rules for users.

1. Information Collection

The App does not collect, upload, or share any personal user data. The App does not access sensitive permissions such as camera, photo library, location, microphone, contacts and network. No private user data will be obtained.

2. Local Data Storage

The pose data collected by users is only stored locally on the user's mobile device and will not be uploaded to any cloud server. After uninstalling the App, all local collected data will be permanently deleted and cannot be recovered.

3. Network Usage

The App works completely offline with no network requests. All pose resources are built into the installation package without traffic consumption.

4. Third-Party Services

This App does not integrate any third-party SDKs, advertising plugins or data analysis tools. There is no third-party data collection, push or advertisement display behavior.

5. Material Copyright Statement

All illustrations in the App are original AI-drawn materials by the developer. There are no real human portraits and no copyrighted images from the Internet. All materials avoid portrait rights and copyright infringement risks, only for personal daily reference.

6. User Rights

Users may delete or uninstall the App at any time without leaving any usage traces. If you have any questions or suggestions, please contact the developer via email.
